Used by https://coveritup.app

## Install

```sh
curl -sLk https://raw.githubusercontent.com/kevincobain2000/cover-totalizer/master/install.sh | sh
```

## Code Coverages

```sh
cover-totalizer coverage {path to clover.xml or coverage.xml file}
```
